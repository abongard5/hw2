JavaScript

Posted by Albert on February 27, 2020

When I first started my Portfolio page, I had no idea what I wanted to do for my JavaScript portion. I looked at a bunch of websites for inspiration, and finally after looking for a while I finally found what I wanted to do. I initially wanted to create a carousel with three pictures, but for some reason I wasn't able to properly implement it. I worked on it for hours, but unfortunately I wasn't able to do it. I was very disappointed after failing for hours, so I did the next best thing I could think of. I added drop down menus for the Technical Skills section. While this isn't as impressive or as pleasing as the image carousel would be, it would have to do. If I allowed more time for myself to work on the Portfolio page, I definitely would've spent more time seeing where I went wrong on the implementation of the image carousel.
